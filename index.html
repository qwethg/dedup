快捷规则
AI 把重复组摘要（文件名 / 来源 / 大小 / 日期，默认不发路径）发给 AI，把自然语言转成勾选规则，服务端强制校验后才勾选——保护文件不会被勾、每组保底留一个，勾选后可一键撤销。

成本预估（以 kimi-k2.6 为例，单次点击）
· 20 组以内 ≈ ¥0.05–0.1
· 60 组（单次上限）≈ ¥0.15–0.3
超过 60 组只处理前 60 组，可分次操作。在设置中开启"发送完整文件路径"会增加约 30–50% 输入成本。实际以 API 控制台账单为准。 把自然语言条件（如"只看 2024 年大于 50MB 的 PDF"）转成筛选条件，只过滤显示的重复组，不改变勾选和扫描数据。消耗极小（单次不足 ¥0.01）。

01它能做什么

        
电脑用久了，同一个文件总会出现在好几个地方。微信里收一份，下载文件夹躺一份，桌面还有一份。

        
归置檔做的，就是把它们找出来——留一份，清掉其余的。也可以反过来，把散在各处的文件按类别收拢到一起。

        
所有事都在本机完成，文件不出门。

      


      

        
02两件事

        
重复清理。按内容把相同的文件分成一组一组，你勾掉多余的副本。默认的精确模式最严格；想查「名字不同、内容一样」的，换内容模式；想认出「副本」「(1)」这类，用相似模式。

        
文件整理。按文件名关键词自动归类，勾好之后一键归档到目标文件夹。实在分不进的「其他」，可以让 AI 看着文件名帮忙归一归。

      


      

        
03上手

        
    
          
  1. 添加要扫描的文件夹。微信、企业微信的目录可以一键识别。
    2. 
          
  2. 点「扫描重复文件」或「整理文件」，等它跑完。
    3. 
          
  3. 用快捷规则勾：保留最新、保留最旧，或保留某个来源。也可以一组一组自己挑。
    4. 
          
  4. 底部操作栏选「移动」或「删除」。移动遇到同名自动加后缀，删除进回收站。
    5. 
          
  5. 结果能导出 JSON / CSV。做过的事，日志里都记着。
    6. 
        

      


      

        
04AI 这事

        
AI 是选配，不填也不影响基本使用。

        
在设置里填一个 OpenAI 兼容的 API 就行。之后可以用大白话让它勾——「保留企业微信的，删其他的」；可以让它按条件筛选；整理页的「其他」也能交给它分类。同一批结果可以连续点好几次，它会接着上次的继续勾，随时能逐次撤销。

        
默认只发文件名、来源、大小、日期，不发完整路径。介意的话，接 Ollama 或 LM Studio 的本地模型，完全离线。每次大概花多少 token，按钮上悬停有预估，通常是几分钱的事。

      


      

        
05几条底线

        
    
          
  • 删除一律进回收站，能反悔。
    • 
          
  • 每组至少留一个，删不光。
    • 
          
  • 文件夹可以标成受保护，里面的文件不会被勾，也不会被碰。
    • 
          
  • AI 勾的结果先过服务端校验才生效，也能一键撤销。
    • 
          
  • 同名文件不覆盖，自动加后缀。
    • 
        

      


      

        
06一些提醒

        
    
          
  • AI 勾完，自己再扫一眼。它快，但判断是你的。
    • 
          
  • 大文件夹扫起来慢，设置里限定文件类型和最小大小，会快不少。
    • 
          
  • 扫描的时候别去动扫描目录里的文件，结果会失真。
    • 
          
  • config.json 和日志都在程序旁边，换电脑之前记得带上。
    • 
          
  • 关掉那个黑色控制台窗口，服务就停了。
    • 
        

        
删文件这事，说到底有风险。回收站给你留了一条回头路——先删，确认没问题，再清空。
