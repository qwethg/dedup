# 归置檔 | Dedup

清理微信 / 企业微信多年堆积的接收文件。

单文件 exe，双击即用。纯本地，免登录，文件不出本机。

## 为什么做

微信、企微用几年，文件全堆在深藏的 `FileStorage` / `WXWork` 目录里。同名文件一堆，不敢删，怕丢重要合同。

通用去重工具能找到重复文件，但不知道哪份来自微信、哪份来自企微。而这恰恰是决定留哪份的依据。

所以做了这个。

## 功能

- **一键发现**微信 / 企微文件夹，不用自己翻目录
- **来源标注**：每个文件标清楚来自微信、企微还是自定义文件夹
- **中文说规则**：打一句“保留企业微信的，删其他的”，AI 转成勾选，你确认后才执行
- **快捷规则**：不用 AI 也行，保留最新 / 最旧 / 指定来源，点一下就好
- **文件整理**：按关键词和 AI 语义分类，可按类别归档
- **删除只进回收站**，可恢复，绝不自动删

## 下载

到 [Releases](../../releases) 下载 exe，双击运行，浏览器自动打开界面。

Windows，无需安装任何依赖。

## 源码运行

```bash
git clone https://github.com/qwethg/dedup.git
cd dedup
pip install -r requirements.txt
python app.py
```

## AI 配置（可选）

设置面板填 API Base、Key、模型名，OpenAI 兼容接口都支持。

- 默认示例是 Moonshot 的 `kimi-k2.6`
- 接 Ollama / LM Studio 本地模型，完全离线
- 隐私开关默认关：AI 只看文件名，看不到路径
- AI 勾选经服务端强制校验：保护文件不勾，每组保底留一个

不配 AI 也能用，快捷规则覆盖大部分场景。

## 安全

- 绝不自动删除，删除 = 回收站
- 只操作你配置过的文件夹
- API Key 只存本地，不回显，不上传
- 所有删除、移动、AI 操作写入本地日志
